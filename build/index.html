    Tables

    Base Table

    Change What happens on Contacts and Accounts? What happens on Household Accounts?
    Modify an Address record marked as Default The change is copied to the Contact's Mailing Address fields or the Account’s Billing Address fields. The change is copied to the Household Account’s Billing Address fields and all Contacts’ Mailing Address fields (except Contacts that have the Do Not Automatically Update box checked).
    Modify an Address record that IS NOT marked as Default The change only affects the Address record you updated. The change affects the Address record you updated. If any Contacts are using this Address (with Do Not Automatically Update box checked), the change is copied over to the Contact's Mailing Address fields.
    Mark an Address record as Default The Address from the record you marked as Default is copied to the Contact's or Account’s Mailing or Billing Address fields. The Address from the record you marked as Default is copied to the Household Account's Billing Address fields and all Contacts’ Mailing Address fields (except Contacts that have the Do Not Automatically Update box checked).
    Create a new Address from the Addresses related list on a Contact or an Account The address IS NOT copied to the Contact's Mailing Address fields or the Account’s Billing Address fields. The Address IS NOT copied to the Household Account’s Billing Address fields.
    Create a new Address from the Addresses related list on a Contact or Account AND mark it as Default The Address from the new record is copied to the Contact's Mailing Address fields or the Account’s Billing Address fields. The Address from the new record is copied to the Household Account's Billing Address fields and all Contacts’ Mailing Address fields (except Contacts that have the Override Address box checked).
    Modify the Billing Address fields on an Account A new Address record is created in the Addresses related list. The new Address is automatically marked as Default Address. A new Address record is created in the Addresses related list. The new Address is automatically marked as Default Address and is copied to all Contacts’ Mailing Address fields (except Contacts that have the Override Address box checked).
